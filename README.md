## Getting started

### Installation
Install all dependencies:
```
yarn setup
```

### Running
To run while connecting with a specific database:
```
yarn cross-env APP_PREFIX=project POSTGRES_DATABASE_NAME=project POSTGRES_HOST_NAME=localhost POSTGRES_PORT=5432 yarn start
```
